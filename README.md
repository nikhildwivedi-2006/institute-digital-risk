
# Institute of Digital Risk – Homepage & Logo Design

This project is created as part of an internship assignment.  
It includes a responsive homepage design and a custom logo for the **Institute of Digital Risk (IDR)**.

## Project Overview

The goal of this project is to design a clean and modern landing page for the Institute of Digital Risk and create a unique logo that represents digital security, risk awareness, and resilience.

## Features

- Responsive homepage design
- Custom SVG logo design
- Clean and modern UI
- Organized folder structure
- Built using basic web technologies

## Technologies Used

- HTML5
- CSS3
- JavaScript
- SVG (for logo design)

## Logo Concept

The logo is designed using SVG and represents the concept of **digital risk and resilience**.

- The cube structure represents a secure digital system.
- The dashed diagonal line represents risk analysis and monitoring.
- The highlighted point represents resilience and control over risk.
- The orange color symbolizes alertness, security awareness, and innovation.
